# Usage

```bash
./appVerCheck.py -u 'AppStoreURL' -txt identifier
```

## Example Use

```bash
./appVerCheck.py -u 'https://itunes.apple.com/jp/app/microsoft-teams/id1113153706?mt=8' -txt teams
```
